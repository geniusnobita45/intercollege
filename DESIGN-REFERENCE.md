# Inner-page design reference

The landing page visual direction remains unchanged. The new registration and faculty pages use general higher-education interface patterns observed on Amity University’s public inner pages:

- breadcrumb-led inner-page hero areas
- clear admission steps and prerequisites
- structured applicant forms
- searchable faculty directory patterns
- detailed educator profile information
- strong admission calls to action

No Amity text, branding, photographs, code or page layout was copied. HLS branding, colours, typography and components remain consistent with the approved website concept.

Reference pages reviewed:

- https://noida.amity.edu/how-to-apply
- https://www.amity.edu/admission/app_form.asp
- https://www.amity.edu/faculty-list-new.aspx
- https://www.amity.edu/gurugram/abs/infrastructure
