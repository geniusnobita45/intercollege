# Content and image sources

This website concept adapts public-facing information and photography from the HLS Inter College website for a modern presentation.

## Public source pages

- Home: https://hlsintercolleges.in/
- About: https://hlsintercolleges.in/about.php
- Contact: https://hlsintercolleges.in/contact.php

## Content used

- College name and Devmanpur/Ghatampur location
- Public phone numbers and email address
- Mission and educational approach
- Campus area and co-educational description
- Public statistics shown on the existing website
- Principal and manager names/qualifications
- Public event names and college gallery images

## Locally packaged photographs

Selected photographs from the public HLS website are stored in `public/images` so the new inner pages do not depend on slow or unavailable remote image requests. They include the college entrance, campus gatherings, student projects, performances, and assemblies.

The photographs are used as:

- the static background on non-landing pages
- the image layer in inner-page hero sections
- visual storytelling cards on registration and faculty pages
- supporting campus imagery on individual teacher profiles

The 240 supplied animation frames remain stored unchanged in `public/frames`.

## Review note

The public website contains some inconsistent historical/establishment figures across pages. This concept deliberately avoids presenting those conflicting figures as fact. The college should verify all copy, statistics, personnel details, and image permissions before publishing this concept as its official website.
